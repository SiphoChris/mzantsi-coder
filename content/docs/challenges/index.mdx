# Challenges

Each challenge here is linked to a channel episode. You don't need to have watched the video
to attempt it — the GitHub repo README has everything you need: the brief, the requirements,
and hints if you get stuck.

## How it works

1. Pick a challenge below
2. Click the GitHub link on the challenge page
3. Read the README — it's your brief
4. Build it, then come back and share what you made
