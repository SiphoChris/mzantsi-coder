## What this challenge is about

JavaScript arrays come with a powerful set of built-in methods that let you transform,
filter, and reduce data without writing manual loops. This challenge gets you comfortable
with the ones you'll use every day.

## What you'll practise

- `map` — transform each item in an array
- `filter` — keep only items that match a condition
- `reduce` — fold an array down to a single value
- `find` and `findIndex` — locate items by condition
- Chaining methods together
