# Resources

A curated collection of tools, references, and reading material that come up regularly
on the channel. No fluff — just things that are actually useful.

## Learning

- [MDN Web Docs](https://developer.mozilla.org) — the reference for HTML, CSS, and JavaScript
- [The Odin Project](https://www.theodinproject.com) — free, project-based web dev curriculum
- [CS50](https://cs50.harvard.edu) — Harvard's free intro to computer science

## Tools

- [Excalidraw](https://excalidraw.com) — quick diagrams and wireframes
- [Ray.so](https://ray.so) — beautiful code screenshots for sharing
- [Regex101](https://regex101.com) — test and understand regular expressions

## South African Dev Community

- [ZATech Slack](https://zatech.co.za) — the SA tech community Slack
- [Dev.to ZA](https://dev.to/t/southafrica) — SA-tagged posts on Dev.to
